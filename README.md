## SIDELOAD
Releases にある house-xxx.dsk ファイルをダウンロード、[Apple\]\[js](https://www.scullinsteel.com/apple2/) などの AppleII 環境で動かしてください。

## 操作
- 文字を入力して `RETURN` で決定
- `←` で一文字削除
